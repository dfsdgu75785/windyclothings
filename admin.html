📊 Google Sheets — Setup Guide

1
Go to sheets.google.com → Create new sheet → name it WINDY Store. Rename Sheet1 tab to Orders, Sheet2 to Reviews. Add the column headers from the guide inside windy-store.html
2
In the sheet: Extensions → Apps Script → delete all existing code → paste the Apps Script code from the comment block inside windy-store.html
3
Deploy → New deployment → Web app. Execute as: Me. Who has access: Anyone. Click Deploy → copy the Web App URL
4
Open windy-store.html → find const SHEET_URL = → paste your URL. Do the same in windy-admin.html → find const SHEET_URL = → paste same URL
5
Done! ✅ Every order placed in the store instantly appears as a new row in your Google Sheet AND in this admin panel. Your whole team just opens the same Google Sheet link to see live orders.
